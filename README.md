# Sugarnope — Meta 광고 성과 대시보드

슈가놉 강미선 Meta 광고계정(ID `2479603099134351`, 비즈니스 Artience)의 **실데이터 기반** 광고 성과 대시보드입니다. Meta Ads API로 직접 조회한 수치를 사용합니다.

## 여는 법

빌드·설치 없이 동작하는 자체완결형 정적 페이지입니다.

- `index.html`을 브라우저로 더블클릭해서 엽니다. (Chart.js를 `vendor/`에 내장해 인터넷 없이도 차트가 그려집니다. 폰트·아이콘만 CDN을 사용합니다.)
- 정적 서버로 열려면 (선택): `npx serve` 또는 임의의 static 서버에서 루트를 서빙.

## 구성

```
Sugarnope/
├─ index.html          대시보드 마크업
├─ assets/
│  ├─ data.js          실측 데이터 (window.SUGARNOPE_DATA) — 여기만 바꾸면 갱신
│  ├─ app.js           렌더링(차트·표·인사이트)
│  └─ styles.css       스타일
├─ vendor/chart.umd.js Chart.js 4.4.1 (내장)
└─ README.md
```

## 대시보드 내용

- **목표 게이지** — 장바구니 전환값 기준 ROAS 911% vs 목표 300%
- **KPI** — 광고비, 장바구니 전환값/건수, ROAS, CTR, CPC, CPM, 도달·빈도
- **일별 추이** — 광고비·전환값·ROAS (6/20–6/23)
- **세부 분석** — 매체 / 지면 / 연령 / 성별별 ROAS·CPA (탭 전환)
- **전환 퍼널** — 픽셀 사이트 전체(페이지뷰→구매)와 단계 전환율
- **핵심 발견 & 실행 제안**

## 측정 주의 (중요)

1. 표시되는 ROAS는 광고 기여 **장바구니 담기** 전환값 ÷ 광고비입니다. 사용자가 정의한 `장바구니+결제시작+결제완료` 합산의 **하한선**입니다.
2. 결제시작·결제완료의 광고 기여 **금액**, 구매 ROAS는 Meta API에서 제공되지 않습니다(이 계정 캠페인의 최적화 이벤트가 '장바구니'이기 때문). 집행 기간 광고 귀속 구매는 사실상 0건입니다.
3. 퍼널 수치는 픽셀 **사이트 전체**(광고+자연 유입) 기준이며 광고 기여분이 아닙니다.
4. `장바구니+결제시작+결제완료` 합산 정의는 동일 매출을 중복 계산하므로, 실집행 판단은 **구매 기준 ROAS**를 메인 지표로 권장합니다.

## 데이터 갱신

새 기간 데이터를 받으면 `assets/data.js`의 숫자만 교체하면 됩니다. 마크업/로직 수정 불필요.

## 한계 — 타 슈가놉 계정 비교

현재 접근 권한에는 슈가놉 계정 1개(`2479603099134351`)와 빈 계정 1개만 있습니다. 다른 슈가놉 계정(1~9 등)은 권한 밖이라 실측 비교가 불가능합니다. Artience 자산 권한을 공유하면 멀티계정 비교 뷰를 추가할 수 있습니다.
